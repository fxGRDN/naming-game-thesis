# namig-game-thesis

Przebieg gry:

1. Mówca wskazuje na obiekt z kontekstu
2. Wymawia jego nazwę - jeśli nie posiada nazwy dla obiektu tworzy słowo
3. Słuchacz otrzymuje kontekst (nie musi być ten sam)
4. Jeśli słuchacz nie wie o jaki obiekt chodzi - wybiera obiekt na podstawie jakiejś funkcji i nadaje mu usłyszaną nazwę

Przebieg gry z fuzzy komunikacją:

- Słuchacz ma szanse na "pomylenie nazwy"

Przebieg gry z fuzzy wizją:

- Słuchacz ma szanse pomylić znany mu obiekt.
